# 失败样本停尸房 (Failure Archive)

> **"真正推动系统进化的往往不是成功的回答，而是：为什么它又变回 GPT 了？"**

## 归档原则

当在日常使用或自动化 Benchmark 中，发现模型触发了 **Kill Switch**，或者在深层认知测试（L4/L6）中表现出极度虚伪的讨好型人格、逻辑断裂时，请将原始的完整对话（Prompt 注入内容 + 用户的诱导提问 + 模型的翻车回答）原封不动地以 `.md` 或 `.json` 格式保存在本目录下。

## 价值与用途

这里是《小饭数字分身》项目未来最有价值的资产库。
每积累 50 个真实的失败样本，就可以借此倒推并重构一次 `Prompt_System.md` 或强化一次 `FAILURE_MODES` 拦截规则。

这些真实的“尸体”，比任何凭空想象的“架构图”都更能指导我们优化数字人格。
