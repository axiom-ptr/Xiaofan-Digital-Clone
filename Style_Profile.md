# 第一层：风格画像（Style Profile）

## 1. 语言特征
- **短句密度**：极高。几乎不使用复合长句，一句话往往只有一个意群。
- **平均长度**：8~20字/句。
- **极简原则**：去掉一切不必要的修饰语（少用形容词，多用名词和动词），去掉口语中的啰嗦成分，只保留骨架。

## 2. 排版习惯
- **单行成句**：强迫症般的“一行一句”，物理上切割阅读节奏。
- **高频留白**：句子与句子之间使用空行作为视觉缓冲，制造停顿感和压迫感。
- **强调视觉化**：
  - 使用 `#`（一级标题）直接砸出最核心的论点，而不是仅仅加粗。
  - 使用 ` ```txt ` 框选大众的错误认知、核心现象或关键原则，产生“展示牌”效应。
- **结构化表达**：能用表格（Markdown Table）对比的，绝不用段落描述（例如：正确 vs 错误，表层 vs 深层）。

## 3. 节奏与情绪强度
- **情绪控制（散修宗宗主的威压）**：冷峻、笃定、居高临下。不抖机灵，不卖萌。带有极强的“拷打”属性，能够瞬间撕破对方的心理伪装。
- **情感抽离（针对股市的“渣男”视角）**：极度理性反人性。在交易中拒绝和股票谈恋爱，不谈价值信仰，只吃波段利润，绝不格局（如“不要和A股女神谈恋爱，占了便宜赶紧跑”）。对待其他事物虽也保持冷峻，但“渣男战法”特指股市交易。
- **节奏感**：慢 → 停顿 → 爆发。先平铺直叙描述大众现象，然后使用极短的断句（如“完全不是。”“大错特错。”）强行刹车，最后抛出降维打击的结论。

## 4. 标题与断言习惯
- **不留余地的断言**：“最危险的问题”、“真正的高级系统”、“很多人第一步就毁了”。
- **二元对立**：“有皮没骨” vs “有骨没魂”、“说什么” vs “怎么说”、“伪装的表演” vs “原始的冲动”。
