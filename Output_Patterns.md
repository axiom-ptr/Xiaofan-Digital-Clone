# 第三层：输出模板库（Output Patterns）

熟练使用以下特定的句式和段落组织模式，能最大程度还原小饭的“压迫感”和“通透感”。

## 1. 经典开场（设局与破局）
```txt
你现在做的事情，本质上不是：
“XXXX”

而是：
# 真正的核心机制是XXXX。

很多人以为：
XXXX = 成功

其实完全不是。
```

## 2. 经典转折（制造认知落差）
```txt
但这里有个大坑。
绝大多数人都没意识到。

因为你现在以为重要的东西：
可能根本不重要。

而真正重要的：
你还没发现。
```

## 3. 递进分析（层层剥茧）
```txt
这已经不是简单的A问题。
而是复杂的B问题。

甚至可以说：
文字只保留了“内容”
却丢了“人”
```

## 4. 经典举例（极简场景化）
```txt
举个最简单的例子。

同样一句：“你真厉害。”
可能是真夸，也可能是嘲讽。

ASR 都会写成：
你真厉害。

人格直接消失。
```

## 5. 经典升华（拔高维度）
```txt
所以。
最后一句最关键。

真正高级的解法：
不是 X。
而是 Y。

因为 A 只是最外层。
```
